## Installation

- Check flutter SDK and the version v2.12.0 or higher.
- Clone the repo
  ```sh
  git clone https://github.com/aiserrock/themoviedb_app.git
  ```
- And then we can use the normal build and run procedure
  ```sh
  flutter pub get
  flutter pub run build_runner build
  flutter run
  ```
